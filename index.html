Real acceleration data. Real cars. No hype.

We’re building a straightforward home for Dragy-verified 0–60 and ¼-mile results. Core values: Truth, Trust, Reliability, Transparency, Thoroughness.

Mission

Make performance information accessible for as many cars as we can test. Present it clearly, keep tests consistent, and separate data from opinion.

What’s coming (MVP)

Results Browser

  • Search/filter by make, model, year, body style.
  • Sort by 0–60 and ¼-mile times.
  • Filter by stock vs modified.

Test Pages

  • 0–60 and ¼-mile results (Dragy proof image).
  • Make/model/year, mods, manufacturer HP/Torque.
  • Conditions (weather/surface) and a link to the run on YouTube.
  • No opinions on the results page.

About & Process

  • How we collect and validate times.

Methods (at a glance)

Principles

  • Consistent setup across vehicles.
  • Dragy-verified runs with evidence.
  • Conditions recorded (weather, surface).
  • Objective data pages; opinions kept separate.

Standard data per test

  • Make / model / year; stock vs. mods listed.
  • 0–60 mph time and ¼-mile time.
  • Manufacturer HP / Torque.
  • Conditions and Dragy screenshot.
  • YouTube link to the run.
